## ❓ FAQ / 常见问题

**Q：俱乐部模式与普通现金桌有什么区别？**  
A：俱乐部模式支持独立俱乐部创建、成员邀请、内部排名、代理分润，以及跨俱乐部联赛。更适合本地化社群运营。

**Q：一个俱乐部最多支持多少会员？**  
A：无硬性上限，取决于服务器配置。建议单俱乐部不超过 5000 活跃会员。

**Q：代理分润可以自定义吗？**  
A：可以。支持多层级、多比例配置，后台可随时调整。

**Q：是否支持多币种 / 真实支付？**  
A：已集成代币系统，可替换为 USD、THB、VND 等真实支付通道。

**Q：交付后提供多久技术支持？**  
A：产品/源码授权提供 **3 个月** 免费技术支持，后续可选年费服务（15% 授权费）。
